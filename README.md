````markdown
### Installation

1. Clone the repository:

   ```bash
   git clone git@github.com:tanvir510/webmanaza-bookshop.git
   ```
````

2. Change to the project directory:

   ```bash
   cd webmanaza-bookshop
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start the development server:

   ```bash
   npm run dev
   ```
